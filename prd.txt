# Finary — AI Personal Finance Tracker

AI-powered finance tracker built with Next.js 14, Zustand and OpenRouter.
Track expenses, visualize spending patterns, set budgets and chat with
your personal AI financial assistant.

## Features
- 📊 Interactive dashboards with Recharts
- 🤖 AI assistant powered by Llama 3.1 (free via OpenRouter)  
- 🎯 Budget tracking with smart warnings
- ✈️ Financial goals with progress tracking
- 📅 Spending heatmap and analytics
- 💾 100% local data (localStorage)

## Tech Stack
Next.js 14 · TypeScript · Tailwind CSS · Framer Motion · Recharts · Zustand

Stórz je fajne to tylko przykla
